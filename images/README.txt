Place the following PNG images in this directory:

  background.png  - (1) Background image
  normal.png      - (2) Default character illustration (normal eyes)
  closed.png      - (3) Character with eyes fully closed (blink)
  half.png        - (4) Character with half-closed eyes (blink transition)
  smile.png       - (5) Smile eye overlay - transparent PNG, shown on top
  mouth.png       - (6) Closed-mouth overlay - transparent PNG
                        (hidden = mouth open, visible = mouth closed)

All images should be the same resolution.
Transparent PNGs (5, 6) are composited on top of the base layers.
